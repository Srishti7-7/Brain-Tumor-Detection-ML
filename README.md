# Brain Tumor Detection using Machine Learning

This project predicts brain tumor type from MRI images using Machine Learning algorithms.

## Dataset
MRI brain images classified into:
- Glioma
- Meningioma
- Pituitary
- No Tumor

## Feature Extraction
- HOG (Histogram of Oriented Gradients)
- GLCM (Gray Level Co-occurrence Matrix)

## Dimensionality Reduction
- PCA (Principal Component Analysis)

## Machine Learning Models
- KNN
- SVM
- Random Forest
- Decision Tree
- Logistic Regression
- Naive Bayes

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve
- AUC Score

## GUI Feature
A Tkinter-based GUI allows users to:
- Train the model
- Upload MRI image anytime
- Predict tumor type

## Installation

Install dependencies:

pip install -r requirements.txt

Run the program:

python brain_tumor_ml.py
